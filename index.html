CSS 선택자 및 적용 방법

CSS 선택자는 해당 요소(태그)에서 서식(Style)을 부여할 대상을 지정하는 것을 CSS 선택이라고 하고, 이 때 사용되는 키워드를 CSS 선택자라고 함

인라인 스타일 방식은 해당 요소에 style="border-bottom:5px solid #333"와 같이 직접 style 속성으로 서식을 지정하는 방법

내부 스타일은 현재 문서의 style 태그 안에 스타일 지정하는 방식으로 현재 문서에만 작성한 스타일을 적용하고자 할 때 사용되는 방식

외부 스타일은 공통되는 스타일을 별도의 css 파일로 작성한 후 link 태그로 스타일 내용을 불러와 연결하는 방식
